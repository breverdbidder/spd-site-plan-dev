# Permit Requirements - SPD

## Arguments
- `$1`: Project name or parcel ID

## Standard Permit Checklist

### Planning & Zoning
- [ ] Pre-application meeting
- [ ] Site Plan Review
- [ ] Variance (if needed)
- [ ] Conditional Use (if needed)

### Building
- [ ] Building Permit
- [ ] Electrical Permit
- [ ] Plumbing Permit
- [ ] Mechanical Permit

### Environmental
- [ ] SJRWMD ERP
- [ ] Tree Removal
- [ ] Wetland Assessment

### Utilities
- [ ] Water Connection
- [ ] Sewer Connection
- [ ] Stormwater

## Timeline Reference

| Phase | Duration |
|-------|----------|
| Pre-application | 2-4 weeks |
| Site Plan Review | 4-8 weeks |
| Building Permit | 2-4 weeks |
| **Total** | 8-16 weeks |

## Fee Estimation

| Permit | Range |
|--------|-------|
| Site Plan | $1,000-$3,000 |
| Building | 1.5% of value |
| Impact Fees | $5,000-$15,000 |
