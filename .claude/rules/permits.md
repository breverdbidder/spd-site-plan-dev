# Permit Rules

## Brevard County Permit Types

### Building Permits
- **Residential New**: New home construction
- **Commercial New**: New commercial buildings
- **Addition/Alteration**: Modifications to existing
- **Demo**: Demolition permits

### Site Development Permits
- **Site Plan**: Required for commercial development
- **Subdivision**: Division of land into lots
- **Variance**: Deviation from zoning requirements
- **Special Exception**: Conditional use approval

## Permit Process Flow

```
Pre-Application → Application → Review → Revisions → Approval → Inspection → CO
```

### Timeline Estimates
| Permit Type | Initial Review | Total Time |
|-------------|---------------|------------|
| Site Plan (Minor) | 2-3 weeks | 4-6 weeks |
| Site Plan (Major) | 4-6 weeks | 8-12 weeks |
| Variance | 6-8 weeks | 10-14 weeks |
| Subdivision | 6-8 weeks | 12-16 weeks |

## Required Documents

### Site Plan Application
- [ ] Application form
- [ ] Site plan drawings (sealed by PE/PLS)
- [ ] Drainage calculations
- [ ] Traffic impact (if applicable)
- [ ] Environmental assessment
- [ ] Fee payment

### Review Comments Response
1. Receive comments within review period
2. Address each comment individually
3. Resubmit revised plans
4. Track resubmittal rounds
