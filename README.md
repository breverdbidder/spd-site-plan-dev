## 12-Stage Pipeline

| Stage | Name | Description | Output |
|-------|------|-------------|--------|
| 1 | **Discovery** | BCPAO data, parcel boundaries, ownership | Property Profile JSON |
| 2 | **Survey Analysis** | Extract dimensions, easements, legal description | Survey Data JSON |
| 3 | **Zoning Analysis** | District, permitted uses, setbacks, density | Zoning Matrix |
| 4 | **Site Constraints** | Wetlands, flood zones, environmental | Constraints Map |
| 5 | **Parking Analysis** | ITE ratios, ADA, local code requirements | Parking Summary |
| 6 | **Traffic Impact** | Trip generation, LOS, turn lanes | Traffic Memo |
| 7 | **Utilities Assessment** | Water, sewer, electric capacity | Utilities Report |
| 8 | **Stormwater Analysis** | Drainage, retention, permit requirements | SW Calculations |
| 9 | **Site Layout** | Building footprints, circulation, landscaping | Conceptual Layout |
| 10 | **ML Feasibility Score** | Cost estimates, timeline, approval probability | Feasibility Score |
| 11 | **Report Generation** | DOCX/PDF deliverables | Final Reports |
| 12 | **Archive** | Supabase storage, version control | Archived Project |
